## A movie database and watchlist app made with **React, Redux and Firebase**.
[Link to the App](https://cineflix-cd.web.app/)


Technologies Used-
- React
- Redux
- Redux Thunk
- React Spring(Animations)
- Firebase(Auth)
- Axios
